Setting up Browser Source

  1. Create a Browser Source in OBS
  2. Copy the View URL using the "Copy for OBS" button above
  3. Paste the URL into the Browser Source's URL field
  4. Set dimensions to match your overlay size (e.g., 800x600)
  5. Enable "Refresh browser when scene becomes active"
